# Full Body Day A

A balanced full-body workout focusing on compound movements.

## Cardio
- Row Machine - 5 min
- Jump Rope - 3 min

## Dynamic Stretch
- World's Greatest Stretch - 2x5 each side
- Leg Swings - 2x10 each
- Arm Circles - 2x10 each

## Prehab
- Band Pull Aparts - 3x15
- Face Pulls - 3x15
- Dead Bugs - 3x10 each

## Push
- Barbell Bench Press - 4x8-10 @ RPE 8
- Incline DB Press - 3x10-12
- Dips - 3x12

## Pull
- Barbell Rows - 4x8-10
- Lat Pulldowns - 3x10-12
- Face Pulls - 3x15

## Hinge
- Romanian Deadlift - 3x10-12 @ 135lbs
- Good Mornings - 3x12

## Lunge/Squat
- Goblet Squats - 3x12 @ 50lbs
- Walking Lunges - 3x10 each
- Bulgarian Split Squats - 3x8 each

## Functional
- Farmer's Walks - 3x40 yards
- Pallof Press - 3x10 each

## Recovery
- Foam Roll - 5 min
- Static Stretching - 5 min
