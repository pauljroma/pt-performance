# Daily Mobility Flow

Quick 20-minute routine for flexibility and joint health.

## Dynamic Stretch
- Cat-Cow - 2x10
- Thread the Needle - 2x8 each
- Hip Circles - 2x10 each
- Ankle Circles - 2x10 each

## Prehab
- Glute Bridges - 3x12
- Bird Dogs - 3x8 each
- Side Lying Clams - 3x12 each

## Functional
- World's Greatest Stretch - 3x5 each
- 90/90 Hip Switches - 3x8 each
- Squat Hold - 3x30s

## Recovery
- Pigeon Pose - 90s each
- Child's Pose - 60s
- Lying Spinal Twist - 60s each
- Deep Breathing - 2 min
