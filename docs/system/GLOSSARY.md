# GLOSSARY – PT Platform Terms

**Purpose:** Ensure humans and agents use terms consistently.

---

## Clinical / PT Terms

- **Pain (0–10):** Patient-reported numeric rating; not a diagnosis.
- **RPE (Rate of Perceived Exertion):** Effort scale (0–10).
- **Return-to-Throw:** A structured progression back to full throwing workload.
- **Phase:** Distinct segment of a program (Rehab, Strength, Maintenance).
- **Session:** A single planned day's work within a phase.

---

## S&C Terms

- **1RM (One Rep Max):** Estimated maximum load for 1 repetition.
- **Strength Zone:** Work typically around 85–95% 1RM.
- **Hypertrophy Zone:** Approx. 70–80% 1RM, higher volume.
- **Endurance Zone:** Approx. 60–70% 1RM, sustained work.

---

## Throwing Terms

- **Bullpen:** Session of structured pitching from mound.
- **Plyo Drills:** Weighted-ball or constraint drills for mechanics/arm care.
- **Hit-Spot%:** Fraction of pitches hitting intended location.
- **Command:** Combination of hit-spot%, consistency, and control.

---

## System Terms

- **Readiness Score:** Composite metric (0–100) derived from pain, adherence, velocity, etc.
- **Flag:** System-level indicator of risk (pain, workload, velocity, adherence).
- **Plan Change Request:** Structured proposal for modifying a program; always goes through Linear.
- **Zone Labels:** Labels in Linear that indicate area of work (data, app, backend, approvals, etc.).

---

## Data Terms

- **Supabase:** Backend (Postgres, Auth, Storage).
- **vw_*:** A database view (e.g., `vw_patient_adherence`).
- **Exercise Log:** A single recorded set of an exercise.
- **Bullpen Log:** A record of a pitching session (velocity, command, pain).

---

## Agent / Control Terms

- **Agent-Control-Plane:** Linear team that orchestrates all agent work.
- **Runbook:** Step-by-step procedure to implement a feature or test.
- **Simulation Scenario:** Synthetic data run to test the system end-to-end.

---

## DoD

- Agents refer to this glossary for any ambiguous term.
- Prompts reference glossary where relevant.
- No contradictory definitions appear in other docs.
