# MVP DEMO SCRIPT – PT App + Agent Platform

**Purpose:** Demonstrate the full capabilities of the platform in 5–7 minutes using the Brebbia example dataset.

---

# 1. SETUP (30 seconds)

Open iPad → Launch PTPerformance

Say:

> "This system blends rehab, performance training, analytics, and AI safety.
> Everything you see ties to real patient data and runs on a governed agent control plane."

---

# 2. PATIENT FLOW (1 minute)

### Step 1 – Today's Session
Open the patient demo mode:

- Show Today's Session ("On-Ramp Week 3 – Day 2")
- Show exercises:
  - Trap Bar Deadlift
  - Shoulder ER
  - Plyo Wall Drills

### Step 2 – Logging a Set
- Log a sample set:
  - 5 reps @ 225 lbs, RPE 6, pain 2.

Explain:

> "Pain and effort automatically update analytics."

### Step 3 – Completion
- Tap Complete → see summary.

---

# 3. THERAPIST DASHBOARD (1.5 min)

### Step 1 – Patient List
Show:
- Readiness Score
- Pain indicator
- Adherence %
- Velocity trend arrow

Say:

> "This is the clinical cockpit: what matters most at a glance."

### Step 2 – Brebbia Details
Tap Brebbia:
- Pain trend chart
- Velocity trend
- Last bullpen session showing command & velo
- Flags: "Velocity drop", "High pain"

Say:

> "This dashboard ingests real bullpen and plyo data — the logic mirrors the sheet you provided."

---

# 4. AI PT ASSISTANT (1.5 min)

Open PT Assistant summary:
- Shows pain ↑
- Shows velo ↓ 4 mph
- Suggests reducing volume

Say:

> "The PT Assistant offers evidence-based guidance.
> It can never diagnose, and all plan changes require human approval."

Press **"Create Plan Change Request."**

Show Linear issue generated:
- Title: "Adjust Week 3 intensity – high pain"
- zone-4b
- Status: In Review

---

# 5. LINEAR + AGENT CONTROL PLANE (1 minute)

Show Linear views:
- "Agent Lane": active tasks
- "Awaiting Approval": zone-4b issues
- "Ready for Agent Execution": next tasks

Say:

> "Agents work under strict governance. No changes bypass Linear."

---

# 6. CODE EXECUTION DEMO (optional, 1 minute)

From Mac:
- Run `/sync-linear`
- Run an agent on "Define Supabase Schema" or "Build Today Session View"

Say:

> "AI executes code against the plan, updates Linear, and maintains safety."

---

# 7. WRAP (20 sec)

Say:

> "This is a next-generation PT platform:
> adaptive, safe, AI-driven, clinically informed, and built for high-performance athletes."
