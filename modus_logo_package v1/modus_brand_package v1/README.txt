LOGO CONCEPT: PROGRESS PATH
---------------------------

The Modus logo features an iconic "M" shape that doubles as an upward
trending progress chart. The design represents:

    - Recovery progress and improvement
    - The dot at the peak symbolizes reaching your goal
    - Upward momentum through the gradient direction

This concept was selected for its meaningful connection to the app's
mission and its excellent legibility at all sizes.


QUICK REFERENCE
---------------

Minimum logo size:     32px (icon), 80px (horizontal logo)
Clear space:           Height of the "o" in modus
Light background:      Use gradient/dark versions
Dark background:       Use white versions
