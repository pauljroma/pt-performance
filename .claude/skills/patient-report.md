# Generate Patient Report

Create therapist-ready patient progress report with summary and charts.

## Trigger

```
/patient-report [patient-id]
```

**Examples:**
- `/patient-report abc123`
- `/patient-report abc123 --email therapist@clinic.com`
- `/patient-report abc123 --format pdf`

## Prerequisites

1. Valid patient ID from `patients` table
2. Patient has session history (at least 1 completed session)
3. Supabase access configured

## Execution Steps

### Phase 1: Fetch Patient Data

Query patient information:

```sql
-- Basic patient info
SELECT
  id, full_name, email, created_at,
  current_program_id, therapist_id
FROM patients
WHERE id = '[patient-id]';
```

### Phase 2: Fetch Session History

```sql
-- Completed sessions with metrics
SELECT
  s.id,
  s.scheduled_date,
  s.completed_at,
  s.total_volume,
  s.session_rpe,
  s.pain_reported,
  p.name as program_name
FROM sessions s
LEFT JOIN programs p ON s.program_id = p.id
WHERE s.patient_id = '[patient-id]'
  AND s.status = 'completed'
ORDER BY s.completed_at DESC
LIMIT 30;
```

### Phase 3: Fetch Readiness Data

```sql
-- Daily readiness scores
SELECT
  date,
  readiness_score,
  sleep_quality,
  energy_level,
  soreness_level,
  stress_level,
  notes
FROM daily_readiness
WHERE patient_id = '[patient-id]'
ORDER BY date DESC
LIMIT 30;
```

### Phase 4: Fetch Pain Logs

```sql
-- Pain tracking history
SELECT
  date,
  body_area,
  pain_level,
  pain_type,
  notes
FROM pain_logs
WHERE patient_id = '[patient-id]'
ORDER BY date DESC
LIMIT 30;
```

### Phase 5: Calculate Metrics

Compute key statistics:

```javascript
// Session adherence rate
const adherenceRate = completedSessions / scheduledSessions * 100;

// Average weekly volume
const avgWeeklyVolume = totalVolume / weeks;

// Average RPE
const avgRPE = sessions.reduce((sum, s) => sum + s.session_rpe, 0) / sessions.length;

// Pain trend
const painTrend = calculateTrend(painLogs);

// Readiness average
const avgReadiness = readinessLogs.reduce((sum, r) => sum + r.readiness_score, 0) / readinessLogs.length;
```

Create markdown report:


### Phase 7: Export (Optional)

#### Export as PDF

```bash
# Using pandoc
pandoc report.md -o patient-report.pdf \
  --pdf-engine=wkhtmltopdf \
  -V geometry:margin=1in

# Or using markdown-pdf
npx markdown-pdf report.md -o patient-report.pdf
```

#### Email to Therapist

```bash
# Using SendGrid edge function
curl -X POST "https://rpbxeaxlaoyoqkohytlw.supabase.co/functions/v1/send-email" \
  -H "Authorization: Bearer [ANON_KEY]" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "[therapist-email]",
    "subject": "Patient Progress Report: [Patient Name]",
    "html": "[Report HTML]",
    "attachments": [{
      "filename": "patient-report.pdf",
      "content": "[Base64 PDF]"
    }]
  }'
```

## Supabase RPC Function (Future)

For production, create an edge function:

```typescript
// supabase/functions/generate-patient-report/index.ts

import { serve } from 'https://deno.land/std@0.177.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'

serve(async (req) => {
  const { patient_id, format } = await req.json()

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!
  )

  // Fetch all data
  const [patient, sessions, readiness, pain] = await Promise.all([
    supabase.from('patients').select('*').eq('id', patient_id).single(),
    supabase.from('sessions').select('*').eq('patient_id', patient_id),
    supabase.from('daily_readiness').select('*').eq('patient_id', patient_id),
    supabase.from('pain_logs').select('*').eq('patient_id', patient_id)
  ])

  // Generate report
  const report = generateReport(patient.data, sessions.data, readiness.data, pain.data)

  return new Response(JSON.stringify({ report }), {
    headers: { 'Content-Type': 'application/json' }
  })
})
```
